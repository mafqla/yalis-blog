2.vite的安装和使用 #

  • 安装vite

    • 使用NPM:

    • npm install vite –g # 全局安装
npm install vite –D # 局部安装

    • 使用yarn：

    • yarn add vite –g # 全局安装
yarn add vite –D # 局部安装

  • 通过vite启动项目

    npx vite

  • 使用vite创建项目

    • 使用 NPM:

    • $ npm create vite@latest

    • 使用 Yarn:

    • $ yarn create vite

    • 使用 PNPM:

    • $ pnpm create vite
