通过本地文件来引入

==注意== 这种引用颜色只能被去色, 不能使用原有的颜色

     '@/assets/css/iconfont.css'  //引入css文件
     
     <span class="iconfont icon-user2"></span>  //span标签使用

4.2.1.2 Symbol 引用 #

直接通过阿里云的绝对路径js文件

==注意:== Symbol引入的 的无须 ttf,woff等字体文件,甚至不需要CSS文件

只需要js和一个通用的.icon 样式

或者自己引入
<script src = 'iconfont.js'></script>


//第二步：加入通用css代码（引入一次就行）：
<style type="text/css">
    .icon {
       width: 1em; height: 1em;
       vertical-align: -0.15em;
       fill: currentColor;
       overflow: hidden;
    }
</style>
//第三步：挑选相应图标并获取类名，应用于页面：
<svg class="icon" aria-hidden="true">
    <use xlink:href="#icon-xxx"></use>
</svg>
